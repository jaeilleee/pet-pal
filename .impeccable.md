## Design Context

### Users
10대~30대 모바일 캐주얼 게이머. 출퇴근/쉬는 시간에 5분씩 플레이. 반려동물에 애정이 있고, 귀여운 캐릭터에 끌리는 사람. 토스 앱인토스 WebView + 모바일 브라우저에서 세로 화면 전용.

### Brand Personality
따뜻한, 통통한, 생동감 있는 (warm, squishy, alive)

### Aesthetic Direction
- **톤**: 3D 클레이/피규어 느낌. 닌텐도 스위치 게임 UI 참고 (동물의 숲, 커비). Pixar 단편 느낌.
- **테마**: 라이트 모드. 따뜻한 크림/베이지 기반 + 포인트 컬러는 코랄/피치.
- **참고**: 네코아츠메, My Talking Tom, 동물의 숲 포켓캠프
- **반참고(NOT)**: 사이버펑크, 네온, 다크 테마, 기업용 대시보드, Material Design 기본값
- **캐릭터**: Pollinations.ai FLUX Schnell로 생성한 3D 렌더 동물 (512x512 PNG, 투명배경)

### Design Principles
1. **캐릭터가 주인공**: UI는 캐릭터를 돋보이게 하는 무대. 버튼/텍스트가 캐릭터보다 눈에 띄면 안 됨.
2. **부드러운 곡선**: 모든 요소가 둥글고 부드러움. 각진 것 없음. 하지만 과도한 radius(32px+)는 피함.
3. **여백으로 호흡**: 정보를 줄이고 핵심만. 빈 공간이 고급스러움.
4. **터치에 반응하는 생동감**: 모든 인터랙션에 미세한 피드백. 스프링 이징.
5. **일관된 깊이 체계**: elevation 3단계로 시각 계층 명확하게.
